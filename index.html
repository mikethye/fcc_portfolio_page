FCC Responsive Web Design Projects

About

This a portfolio of four web design projects as required by the FCC Responsive Web Design Certification. There are a number of user story requirements for each project and each is highly specific in use of; class, id, flex, anchor tags, etc. Each project is an attempt to apply all of the skills, principles, and concepts a student learns within the FCC Responsive Web Design coursework including; HTML, CSS, Responsive Visual Design and Accessibility.
